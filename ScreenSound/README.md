# Programa Screen Sound 3.0

Um sistema para registrar bandas, álbuns e avaliações musicais.

Permite ao usuário adicionar bandas, registrar álbuns, exibir a lista de bandas, avaliar bandas e visualizar detalhes, incluindo a média das avaliações.
